## Crypto App




### Depedencies
   - antd
   - millify
   - moment
   - styled-components
   - react-redux
   
   
### API used 
- coinranking to display different cryptocurrencies and 
- bing-news-search to display news


## To run the project:

### npm i
install node modules


### npm start
Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
